# TTS Converter

Windows 기본 음성 엔진(SAPI)을 이용한 텍스트 음성 변환 프로그램입니다.

# 기존 프로그램 형태
- 초기 버전은 Python 스크립트(`audio_creator.py`)를 PyInstaller로 패키징한 `audio_creator.exe` 형태였습니다.
- 텍스트를 음성으로 바꿀 때 `gTTS`를 사용했고, 이후 `pydub`로 음량과 속도를 조정한 뒤 `mp3`로 저장하는 구조였습니다.
- 실행 시 `script` 폴더의 `.txt` 파일을 읽고, 중간 음성 파일을 만든 뒤 최종 결과를 `audio_data` 폴더에 저장하는 방식이었습니다.

# 기존 버전의 문제점
- `gTTS` 사용 특성상 외부 서비스 의존성이 있어 실행 환경에 따라 안정성이 떨어질 수 있었습니다.
- `pydub` 기반 후처리는 추가 오디오 처리 환경에 영향을 받아 배포형 실행 파일에서 깨지기 쉬웠습니다.
- `PyInstaller` 단일 exe 방식은 압축 해제, 권한, 실행 위치 문제로 일부 환경에서 실행 실패가 발생할 수 있었습니다.
- 숫자 발음 제어가 어려워 `3`을 `셋`처럼 읽거나, `제3자`를 자연스럽지 않게 읽는 문제가 있었습니다.

# 현재 구조
- 현재 버전은 Python 실행 환경 없이 동작하도록 Windows 기본 음성 엔진(SAPI) 기반 PowerShell 스크립트로 다시 구성했습니다.
- 실행 파일 대신 `audio_creator.cmd`가 `audio_creator.ps1`을 호출하는 구조입니다.
- 출력은 호환성과 단순성을 위해 `8kHz mono .wav` 형식으로 저장합니다.
- 숫자 읽기 보정을 추가해 항목 번호와 `제3자`, `제3조`, `제1항` 같은 표현을 더 자연스럽게 읽도록 보정합니다.

# 실행 방법
1. `audio_creator.cmd`를 실행합니다.
2. 같은 폴더 안의 `script` 폴더에 변환할 `.txt` 파일을 넣습니다.
3. 실행이 끝나면 `audio_data` 폴더에 같은 이름의 `.wav` 파일이 생성됩니다.

# 폴더 구조
- `audio_creator.cmd`: 실행용 배치 파일
- `audio_creator.ps1`: 실제 변환 로직
- `script`: 입력 텍스트 파일 폴더
- `audio_data`: 생성된 음성 파일 폴더

# 동작 방식
- `script` 폴더의 모든 `.txt` 파일을 순서대로 읽습니다.
- Windows에 설치된 한국어 음성이 있으면 우선 사용합니다.
- 한국어 음성이 없으면 기본 음성으로 대체합니다.
- 결과 파일은 `audio_data` 폴더에 `8kHz mono .wav` 형식으로 저장합니다.

# 주의 사항
- 이 버전은 Python이나 ffmpeg가 없어도 실행됩니다.
- 음성 품질과 발음은 Windows에 설치된 음성 엔진에 따라 달라집니다.
- 텍스트 파일은 UTF-8 또는 기본 Windows 인코딩(예: CP949)으로 저장해 두면 됩니다.
